ConeOpt
=======
Counterfactual Explanations with Optimization

Installation
------------

1. Create a Python env. For example,

.. code-block:: bash

   conda create -p ./conda_env python-3.6 --yes

2. To install **ConeOpt** package, use:

.. code-block:: bash

    pip install -r requirements.txt

2. Run notebook.

.. code-block:: bash

    jupyter notebook
